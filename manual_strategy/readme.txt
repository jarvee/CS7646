Part 1: Technical Indicators
Run indicator.py to get the value for each indicators. You may change the start date, end date, symbol and lookback parameters in the testcode part to get different value of the indicators.

Part 2: Best Possible Strategy
Run BestPossibleStrategy.py. It includes marketsimcode.py, which is used to calculate the status of portfolios. You may change the start date, end date, symbol parameters in the testcode part to get different value of the portfolios.


Part 3: Manual Rule-Based Trader
Run ManualStrategy.py. It includes marketsimcode.py and indicators.py. You may change the start date, end date, symbol parameters in the testcode part to get different value of the portfolios.
